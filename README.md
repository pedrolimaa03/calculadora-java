# 🧮 Calculadora em Java

## 📌 Descrição
Calculadora desenvolvida em Java com operações básicas utilizando métodos.

## 🚀 Funcionalidades
- Soma
- Subtração
- Multiplicação
- Divisão com validação

## 🛠️ Tecnologias
- Java

## 📚 Aprendizados
- Criação de métodos
- Reutilização de código
- Estruturas condicionais

## ▶️ Como executar
Compile: javac MetodosCalculadora.java

Execute: java MetodosCalculadora.java
